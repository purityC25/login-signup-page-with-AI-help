# Static Login and Signup UI

## Overview

This project is a simple static authentication interface built using HTML and CSS.

It was created as part of the Moringa AI Capstone Project to demonstrate how generative AI prompts can be used to learn, build, and deploy a beginner-friendly web project.

The application includes:

* A login page
* A signup page
* A clean and simple user interface
* Deployment as a static website using Render

---

## Features

* Login form UI
* Signup form UI
* Simple and clean design
* Navigation between pages
* Deployed online using Render

---

## Technologies Used

* HTML5 for structure
* CSS3 for styling and layout
* Render for static site hosting
* Git and GitHub for version control and deployment

## Common Issues and Fixes

### CSS Not Loading

Ensure the CSS file is properly linked:

```
<link rel="stylesheet" href="style.css">
```

---

### Navigation Not Working

Check that file names are correct and links are properly written:

```
<a href="signup.html">Sign Up</a>
```

---

### Deployment Issues

Ensure:

* The GitHub repository is public
* The Render project is connected to the correct repository
* The publish directory is set to the root (.)
